# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Sobre o projeto

**Claudete** é um repositório de testes e experimentos com Claude Code. O objetivo é explorar o fluxo de trabalho com Claude Code, incluindo criação de repositórios, commits e pull requests.

## Estrutura

O repositório está em estágio inicial — atualmente contém apenas documentação (`README.md`). Conforme experimentos forem adicionados, este arquivo deve ser atualizado com os comandos e convenções relevantes.

## Convenções de branch

- Branch principal: `master`
- Branches de feature seguem o padrão `melhoria/<descricao>` (ex: `melhoria/readme-descricao`)

## Commits

Mensagens de commit seguem o padrão **Conventional Commits** em português:
- `docs:` — alterações em documentação
- `feat:` — novas funcionalidades
- `fix:` — correções de bugs
- `refactor:` — refatorações sem mudança de comportamento
